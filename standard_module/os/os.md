# os

## 获取当前工作目录
```python
os.getcwd()
```
## 切换当前工作目录
```python
os.chdir(path)
```
## 列出指定目录下的所有文件和目录
```python
os.listdir(path)
```
## 创建目录
```python
os.mkdir(path)
```
## 删除目录
```python
os.rmdir(path)
```
## 创建文件
```python
os.remove(path)
```
## 路径拼接
```python
os.path.join(path1, path2)
```
## 判断路径是否存在
```python
os.path.exists(path)
```
## 判断路径是否是文件
```python
os.path.isfile(path)
```
